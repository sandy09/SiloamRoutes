SETUP


Save folder into your desired path

Be sure you are running files in python2.7

Open Terminal and install proper libraries:
pip install urllib2, 
pip install json
pip install folium
pip install plotly
pip install webbrowser
pip install string
pip install os
pip install urllib
pip install gpxpy 
pip install gpxpy.gpx
pip install geopy
pip install pandas 
pip install osmread
pip install conda
conda install gdal           if you have issues try:
							conda install -f jpeg=8
pip install shapely
pip install matplotlib
pip install math
pip install sys


After installing all libraries run in terminal from the SiloamRoute directory:
python SiloamRoutes.py
